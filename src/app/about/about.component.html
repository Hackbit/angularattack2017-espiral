About Ecopoints in Málaga

Ecopoints in Málaga shows information about ecopoints in the city of Málaga (Spain): electric car chargers, photovoltaic street lights, wind power generators, etc. This information is open and free and it is provided via API by Malaga's town hall.

The ecopoints are shown in a map. You can see all of them or filter by type using a dropdown menu. Moreover you can see the information about ecopoints in a table (Kendo UI grid).

  • Google Maps API
